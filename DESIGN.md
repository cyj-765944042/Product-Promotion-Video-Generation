# DESIGN.md

## 项目与用户画像
AI商家带货视频生成工具，面向电商商家、带货主播，提供一键生成带货短视频的能力。

## 品牌与视觉方向
- **风格关键词**：简约、柔和、现代、专业
- **主色调**：淡紫色系（柔和、信任感）
- **整体风格**：浅色系、米白色背景、柔和的紫色点缀

## Design Tokens

### 色彩
- **背景色**：#F8F7F5（浅米白色）
- **主色调**：#D4C2F6（淡紫色）
- **按钮主色**：#B999F3（紫色按钮）
- **辅助色**：#7A7A7A（深灰色图标）
- **选中背景**：#ECE6F7（淡紫色浅色调）
- **卡片背景**：#F1F3F5（浅灰白色）
- **输入框背景**：#FFFFFF（白色），边框#E5E5E5

### 文字颜色
- **标题文字**：#333333（深灰色）
- **辅助文字**：#666666（中灰色）
- **时间/次要文字**：#999999（浅灰色）

### 字体
- 字体族：系统默认字体栈

### 布局与响应式
- 最大宽度：`max-w-4xl`
- 内边距：`px-4 py-8`
- 间距：使用Tailwind的间距系统（gap-2, gap-4等）

### 组件规范
- 卡片：浅灰白色背景（#F1F3F5）
- 按钮：淡紫色填充（#D4C2F6或#B999F3），深灰色文字
- 输入框：白色背景，浅灰色边框，圆角
- 侧边栏选中项：淡紫色浅色调背景（#ECE6F7）

### 图标
- 使用 Lucide React 图标库
- 主要图标：Video, Upload, Sparkles, FileText, Download, Loader2, CheckCircle2, Circle
